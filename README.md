# ママログ（Mamalog）

ワンオペ育児中のママ向けプロトタイプ Web アプリです。家族の予定・タスク・気持ちを、スマホ風レイアウト（幅約 390px）で一覧できます。

## 必要な環境

- Node.js 18 以上  
- npm（または Yarn / pnpm など互換ツール）

## トップページで表示される画面

`/`（`src/app/page.tsx`）は **`MamalogApp`** を読み込み、**ホーム**タブを初期表示にしています。同じ画面構成は**カレンダー**タブからも開けます。

1 本の縦スクロール内に、次がまとまって表示されます。

- **＋ 予定を追加**（プレビュー用。押すとブラウザのアラート）
- **月／週／リスト**の切り替え
- **人ごとのタブ**（横スクロール・メンバー色のドットと予定一覧が連動）
- 月グリッド／週ストリップ／リストビューと、**選択日の予定**
- **タスク管理**（今日のタスク／すべて、チェックボックス、タスク追加ボタン）

## セットアップと起動

プロジェクトルートへ移動してから（フォルダ名は環境によります）、依存関係を入れます。

```bash
npm install
```

開発サーバーを起動します。

```bash
npm run dev
```

ターミナルに `✓ Ready on http://localhost:3000` のような表示が出たら、ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

### `npm: command not found` のとき

開発用 PC に **Node.js（npm 同梱）** が入っていません。次のいずれかでインストールし、新しいターミナルで `node -v` と `npm -v` が通ることを確認してから、もう一度 `npm install` / `npm run dev` を実行してください。

- [Node.js 公式 LTS](https://nodejs.org/)
- Homebrew（例）: `brew install node`

## ビルド

```bash
npm run build
npm start
```

## 構成

| 項目 | 内容 |
| --- | --- |
| フレームワーク | Next.js 15 App Router |
| 言語 | TypeScript |
| スタイル | Tailwind CSS |
| 状態 | useState とローカルデータのみ（外部 API／DB は未接続） |

主要 UI は `src/components/` に分割しています。アプリ状態は `MamalogApp.tsx` が集約し、選択中タブ・カレンダー表示モード・メンバー／日付の選択などを保持します。
